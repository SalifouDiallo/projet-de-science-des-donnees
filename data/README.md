# Données – Comptages cyclistes (TfL)

- **Source :** Transport for London (TfL) – Open Data Service  
- **URL :** https://cycling.data.tfl.gov.uk/  
- **Licence :** Open Government Licence (OGL)  
- **Date d’accès :** 21 octobre 2025  
- **Fichier :** 2024 W1 spring-Inner-Part1.csv  
- **Dimensions :** 870 144 lignes × 11 colonnes  

## Variables principales
- `Wave` : identifiant de la campagne de mesure (printemps 2024)
- `SiteID` : code du site de comptage
- `Date` : date de la mesure
- `Weather` : conditions météo (Dry/Wet)
- `Time` : créneau horaire
- `Day` : jour de la semaine (Weekday/Weekend)
- `Round` : sous-période (A/B)
- `Direction` : sens de déplacement (Northbound/Southbound)
- `Path` : type de voie (Carriageway, Track, etc.)
- `Mode` : type de vélo (Cargo, Classic, e-bike, etc.)
- `Count` : nombre de cyclistes comptés dans le créneau
